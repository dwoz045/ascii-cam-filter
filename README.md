# ascii-cam-filter
A real-time ASCII camera filter written in Python. Captures live video from webcam and renders in terminal based on relative intensity of pixels in the feed. 

## Requirements
- Python 3.8+
- OpenCV
- A terminal that supports ANSI escape codes
